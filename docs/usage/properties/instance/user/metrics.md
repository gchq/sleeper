## Instance Properties - Metrics - User Defined

The following instance properties relate to metrics.

| Property Name                                     | Description                                                                                                                                                                                                                                  | Default Value | Run CDK Deploy When Changed |
|---------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------|-----------------------------|
| sleeper.cloudwatch.table.metrics.namespace        | The CloudWatch namespace to publish table metrics to.                                                                                                                                                                                        | Sleeper       | true                        |
| sleeper.lambda.table.metrics.concurrency.reserved | The reserved concurrency for the table metrics lambda.<br>See reserved concurrency overview at: https://docs.aws.amazon.com/lambda/latest/dg/configuration-concurrency.html                                                                  |               | true                        |
| sleeper.lambda.table.metrics.concurrency.max      | The maximum concurrency allowed for the table metrics lambda.<br>See maximum concurrency overview at: https://aws.amazon.com/blogs/compute/introducing-maximum-concurrency-of-aws-lambda-functions-when-using-amazon-sqs-as-an-event-source/ |               | true                        |
| sleeper.run.table.metrics.batch.size              | The number of tables to calculate metrics for in a single invocation. This will be the batch size for a lambda as an SQS FIFO event source. This can be a maximum of 10.                                                                     | 1             | true                        |
| sleeper.run.table.metrics.offline                 | Whether to calculate table metrics for offline tables.                                                                                                                                                                                       | false         | false                       |
| sleeper.dashboard.time.window.minutes             | The period in minutes used in the dashboard.                                                                                                                                                                                                 | 5             | true                        |
